# 📊 Sales Forecasting Dashboard | Power BI

## Overview

An end-to-end Business Intelligence project developed in Power BI to analyze historical sales performance and forecast future revenue using interactive dashboards, advanced DAX, and time intelligence techniques. The solution transforms raw sales data into actionable business insights through executive-level reporting.
